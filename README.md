## 🌟 About the Project

**CSS Art Generator** is a sleek, free, and open-source browser utility for front-end developers and designers. 

It allows you to manipulate elements visually in real-time and instantly export the generated CSS code. Whether you're building a smooth gradient background, a dynamic shape, or experimenting with continuous CSS animations—this tool creates production-ready code in one click! Built entirely with pure HTML, CSS, and Vanilla JavaScript. **Zero dependencies needed.**

## 🚀 Features

- **💠 Dynamic Shapes:** Generate Circles, Squares, and SVG-like Blobs instantly. Tweak Size and Border Radius dynamically.
- **🌈 Vibrant Gradients:** Support for Linear, Radial, and Conic gradients. Modify angle and blend dual-color stops.
- **✨ Glowing Shadows:** Precisely dial in `box-shadow` values like Blur radius, Spread radius, and Shadow Colors for ultra-slick glows.
- **🌀 Keyframe Animations:** Spin, Pulse, Float, and Bounce. Control infinite animation durations natively, with play/pause toggles!
- **🌓 Adaptive Themes:** Stunning Dark Mode (Glassmorphism design) & clean Light Mode built-in.
- **📋 1-Click Copy:** Grabs all the generated inline CSS in your clipboard cleanly.

## 🛠️ Built With

- **HTML5** (Semantically Structured & SEO Optimized)
- **Vanilla CSS3** (Custom Properties, Flexbox/Grid, Transform & Animations)
- **Vanilla JavaScript** (DOM Manipulation & Clipboard API)

## 🏃 Getting Started

To run this tool locally, simply clone the repository and open the HTML file. No complicated build steps or package managers required!

```bash
# Clone the repository
git clone https://github.com/rizvee/css-art-generator.git

# Navigate into the directory
cd css-art-generator

# Open index.html in your favorite browser
start index.html  # Windows
open index.html   # Mac
xdg-open index.html # Linux
```

## 🎨 Why Use This Tool?

Manually writing complex `box-shadow` layers or tricky background gradients like `repeating-conic-gradient` can be tedious. This playground provides an instantaneous visual representation of your configurations—so you spend less time guessing syntax and more time building beautiful UI components.
